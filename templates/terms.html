4. Privacy & Data Encryption

We prioritize your privacy and data security across all features:

  • Personal Note Encryption: Personal notes are encrypted at rest using per-user Fernet keys (AES-128-CBC + HMAC-SHA256). Platform administrators cannot read your private notes.
  • Bond Journal & QotD Encryption: Shared bond journals and Question of the Day responses are encrypted at rest using unique per-bond Fernet keys.
  • Direct Message Encryption: Direct messages are encrypted at rest using envelope encryption keys.
  • Whisper Mode Non-Persistence: Whisper Mode messages exist only in temporary memory with automatic Time-To-Live (TTL) expiration. They are never written to permanent message storage, never included in backups, and are permanently erased upon session expiration.
  • PIN-Locked Notes: Sensitive notes can be locked behind a user PIN. The PIN is securely hashed. Lost PINs cannot be recovered by administrators.
  • Security Infrastructure: Enforced HTTPS with HSTS, Content Security Policy (CSP), and XSS protection headers.
  • No Data Sales: We never sell, trade, or rent your personal information to third parties.

5. Shared Content & Collaboration

  • Screenshot Alerts: Whisper Mode incorporates best-effort browser screenshot and window blur detection. Attempting to capture or record a Whisper session triggers a notification to both participants.

7. Bonds & Partner Connections
